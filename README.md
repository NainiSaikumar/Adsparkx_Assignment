# Persona-Adaptive Customer Support Agent

## Introduction

This project is a customer support application developed using Python and Streamlit. The main objective of this project is to identify the type of user and provide suitable responses based on their needs. The application also retrieves information from a small knowledge base and generates responses using Gemini AI.

## Features

* Detects user persona
* Retrieves information from knowledge base documents
* Generates AI-based responses
* Detects situations that require escalation
* Provides a human handoff summary
* Simple and user-friendly interface

## Technologies Used

* Python
* Streamlit
* Gemini AI
* Python Dotenv

## Project Files

* app.py - Main application file
* data folder - Knowledge base documents
* requirements.txt - Required libraries
* .env - API key configuration

## How to Run the Project

1. Create a virtual environment

python -m venv venv

2. Activate the virtual environment

venv\Scripts\activate

3. Install required packages

pip install -r requirements.txt

4. Run the application

streamlit run app.py

## Conclusion
This project demonstrates the use of AI in customer support systems. It combines persona detection, knowledge retrieval, response generation, and escalation handling in a single application.
